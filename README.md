# Description 📖
The manual, first-come, first-served system for using private carrel rooms at UTM's Perpustakaan Sultanah Zanariah frequently inconveniences students who must physically visit the library without knowing if spaces are available. To address this inefficiency and the lack of usage tracking, a Library Reservation System is being developed that allows users to remotely check availability and reserve rooms in advance. To ensure fair and organized space utilization, this upgraded system will also implement a penalty feature that temporarily suspends booking privileges for users who fail to show up for their reservations more than three times.
